# calculadora-react

Calculadora simple creada con React + Vite.

Características:
- Operaciones básicas: suma, resta, multiplicación (×), división (÷), decimales.
- Interfaz en español.
- CI: tests y despliegue automático a GitHub Pages con GitHub Actions.

Cómo ejecutar localmente:
1. Instala dependencias:
   ```bash
   npm install
   ```
2. Levanta en desarrollo:
   ```bash
   npm run dev
   ```
3. Construir:
   ```bash
   npm run build
   ```
4. Ejecutar tests:
   ```bash
   npm run test
   ```

Despliegue en GitHub Pages:
- El pipeline se ejecuta automáticamente en cada push a la rama `main`.
- La acción construye y despliega el contenido de `dist/`.
- URL del sitio: `https://100495821.github.io/calculadora-react/` (puede tardar unos minutos la primera vez).
